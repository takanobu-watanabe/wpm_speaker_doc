プライバシーポリシー

最終更新日: 2026年4月22日

「WPM英語スピーキング」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリが収集する情報とその利用方法について説明します。

1. 収集する情報

1.1 アプリ利用識別子

  • アプリの利用状況を管理するための識別子（ランダムに生成されるユーザーID）
  • 個人を特定できる情報は含まれません

1.2 アカウント情報（任意）

複数の端末で購入内容を利用するために、ユーザーが任意で「Sign in with Apple」でサインインした場合のみ、以下の情報を取得します：

  • ユーザー識別子（Sign in with Apple が発行する匿名化されたID）

サインインは任意であり、本アプリの機能はサインインなしでもご利用いただけます。サインインしない場合、上記の情報は取得されません。

1.3 音声・テキストデータ

  • スピーキング練習時の音声認識によるテキストデータ
  • 無料機能のスピーキング練習における音声データは、デバイス上でのみ処理され、サーバーには保存されません
  • プレミアム会員のAI評価機能を利用する場合のみ、より高精度な書き起こしのため、録音した音声データ（最長180秒）が一時的に OpenAI Whisper API に送信されます。送信された音声は書き起こし処理にのみ用いられ、サーバーには保存されません

⚠️ 重要なご注意: スピーキング練習で発話された内容は、音声認識によりテキスト化され、AI評価のため Google Gemini API に送信されます。プレミアム会員のAI評価利用時は、音声データそのものも OpenAI Whisper API に送信されます。ご自身やご家族・ご友人等の氏名、住所、電話番号、メールアドレス、クレジットカード番号、その他の個人情報・機密情報を発話しないでください。利用者が発話した内容に起因する情報漏えい等について、開発者は責任を負いません。

1.4 購入情報

  • サブスクリプションの購入履歴（加入状況、開始日、更新状態）
  • AI評価機能の利用回数（1日あたりの上限管理のため）

2. 情報の利用目的

収集した情報は以下の目的で利用します：

  • アプリの機能提供・状態管理
  • スピーキング内容のAI評価
  • サブスクリプションの管理
  • 不正利用の防止

収集した情報を広告やマーケティング目的で利用することはありません。また、第三者による追跡（トラッキング）も行いません。

3. 第三者サービス

本アプリは以下の外部サービスを利用しています：

  • Firebase（Google LLC）: 認証、利用状況管理、バックエンド処理。
  • Google Gemini API（Google LLC）: スピーキング内容のAI評価。テキストデータのみ送信されます。
  • OpenAI API（OpenAI, L.L.C.）: プレミアム会員のAI評価利用時のみ、より高精度な音声書き起こし（Whisper）のため、録音音声データを送信します。OpenAI のデータ利用方針に基づき、API 経由で送信された音声はモデル学習には用いられません。
  • RevenueCat Inc.: サブスクリプション管理。
  • Apple Inc.: Sign in with Apple、App Store決済。

これらのサービス提供者以外の第三者に個人情報を販売・提供することはありません。

4. データの保存

  • 練習履歴は、ご利用中の端末上にのみ保存されます
  • AI評価のために送信されたテキストデータは、本アプリのサーバーには保存されません
  • 利用回数・サブスクリプション状態などのデータは、Firebase に安全に保存されます（SSL/TLS による暗号化通信）
  • データは、アプリを継続してご利用いただく間、または法令上必要な期間保持されます

5. データの削除

以下のいずれかの方法でデータを削除できます：

  • アプリ内から削除: 「このアプリについて」→「プライバシーとデータ」→「アカウントを削除」をタップすると、Firebase に保存されたユーザーデータおよび利用回数データが削除されます
  • アプリのアンインストール: 端末上の練習履歴が削除されます
  • お問い合わせによる削除: 上記の方法で削除できない場合は、下記の連絡先までお問い合わせください

サブスクリプション加入中の方は、アカウント削除前または後に別途 iOS の「設定」→「Apple ID」→「サブスクリプション」から解約が必要です。

6. ユーザーの権利

ユーザーは以下の権利を有します：

  • 自身の個人情報の開示を請求する権利
  • 誤った情報の訂正を請求する権利
  • 個人情報の削除を請求する権利
  • 個人情報の利用停止を請求する権利

これらの請求を行う場合は、下記の連絡先までお問い合わせください。

7. セキュリティ

本アプリは、収集した情報を適切に保護するため、以下のセキュリティ対策を講じています：

  • サーバーとの通信は SSL/TLS による暗号化
  • Firebase Authentication による安全なユーザー認証
  • アクセス制御による不正アクセスの防止

8. お子様のプライバシー

本アプリは13歳未満のお子様を対象としていません。13歳未満のお子様から意図的に個人情報を収集することはありません。

9. プライバシーポリシーの変更

本ポリシーは必要に応じて更新されることがあります。重要な変更があった場合は、本ページの最終更新日を更新してお知らせします。本アプリを継続してご利用いただくことにより、更新後のポリシーに同意いただいたものとみなします。

Privacy Policy

Last Updated: April 22, 2026

"WPM English Speaking" (the "App") respects user privacy and is committed to protecting personal information. This Privacy Policy describes the information the App collects and how it is used.

1. Information We Collect

1.1 App Usage Identifier

  • A randomly generated user identifier used to manage app usage
  • Contains no personally identifiable information

1.2 Account Information (Optional)

Only if you choose to sign in with "Sign in with Apple" to access purchased content across multiple devices, we obtain the following:

  • User identifier (the anonymized ID issued by Sign in with Apple)

Sign-in is optional, and the App's features can be used without signing in. If you do not sign in, the above information is not collected.

1.3 Voice & Text Data

  • Text data from speech recognition during speaking practice
  • For free features, voice data is processed only on the device and is not stored on servers
  • For Premium members using the AI evaluation feature, the recorded audio (up to 180 seconds) is temporarily transmitted to the OpenAI Whisper API to produce a higher-accuracy transcript. The transmitted audio is used solely for transcription and is not retained on servers

⚠️ Important Notice: Content spoken during speaking practice is converted to text via speech recognition and sent to the Google Gemini API for AI evaluation. When Premium members use AI evaluation, the audio data itself is also sent to the OpenAI Whisper API. Please do not speak personal or confidential information such as your own or others' names, addresses, phone numbers, email addresses, credit card numbers, or any other sensitive information. The developer is not responsible for any information leakage resulting from the content spoken by the user.

1.4 Purchase Information

  • Subscription purchase history (enrollment status, start date, renewal status)
  • Usage count for AI evaluation (to enforce daily limits)

2. Use of Information

The collected information is used for the following purposes:

  • Providing and managing app functionality
  • AI evaluation of speaking content
  • Subscription management
  • Preventing fraudulent use

We do not use collected information for advertising or marketing purposes, nor do we engage in third-party tracking.

3. Third-Party Services

The App uses the following external services:

  • Firebase (Google LLC): Authentication, usage management, backend processing.
  • Google Gemini API (Google LLC): AI evaluation of speaking content. Only text data is transmitted.
  • OpenAI API (OpenAI, L.L.C.): Used only when Premium members invoke AI evaluation, to send recorded audio for higher-accuracy transcription via Whisper. Per OpenAI's API data usage policy, audio submitted via the API is not used for model training.
  • RevenueCat Inc.: Subscription management.
  • Apple Inc.: Sign in with Apple, App Store payments.

We do not sell or provide personal information to any third parties other than these service providers.

4. Data Storage

  • Practice history is stored only on the device in use
  • Text data sent for AI evaluation is not stored on our servers
  • Data such as usage count and subscription status is securely stored in Firebase (with SSL/TLS encrypted communication)
  • Data is retained for as long as you continue to use the app or as required by law

5. Data Deletion

You can delete your data using any of the following methods:

  • Delete in-app: Navigate to "About This App" → "Privacy & Data" → "Delete Account" to remove the user data and usage count data stored in Firebase
  • Uninstall the App: Practice history on the device will be deleted
  • Contact us: If you cannot delete through the methods above, please contact us using the information below

If you have an active subscription, please cancel it separately via iOS Settings → Apple ID → Subscriptions, either before or after account deletion.

6. User Rights

Users have the following rights:

  • The right to request disclosure of their personal information
  • The right to request correction of incorrect information
  • The right to request deletion of personal information
  • The right to request suspension of use of personal information

To exercise these rights, please contact us using the information below.

7. Security

The App takes the following security measures to appropriately protect the collected information:

  • Communication with servers is encrypted via SSL/TLS
  • Secure user authentication via Firebase Authentication
  • Access control to prevent unauthorized access

8. Children's Privacy

The App is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

This policy may be updated as necessary. If there are significant changes, we will notify you by updating the "Last Updated" date on this page. Your continued use of the App constitutes your agreement to the updated policy.
